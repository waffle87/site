C Utilities

The leetcode.h header file contains various includes, standard Leetcode structs (ListNode, TreeNode), and macros for things like calculating the size of an array (in a portable manner):

#define IS_ARRAY(value) \
      (!__builtin_types_compatible_p(typeof((value)), typeof(&(value)[0])))

      #define ARRAY_SIZE(array) \
      (__builtin_choose_expr(IS_ARRAY((array)), \
      sizeof((array)) / sizeof((array)[0]), (void)0))

      // foo.c
      int arr[] = {1, 2, 3};
      int foo = func(arr, ARRAY_SIZE(arr));

The leetcode.c file contains helper functions to fill ListNode and TreeNode data structures with given problem input, and also print data returned by such functions.

ListNode Data Type

int main() {
      int input[] = {0, 3, 1, 0, 4, 5, 2, 0};
      struct ListNode *head = listnode_build(input, ARRAY_SIZE(input));
      listnode_print(head);
      listnode_free(head);
      }

TreeNode Data Type

Note that in the following, a value of -1 denotes a "null" leaf. If a problem's input depends on this value, use treenode_create() to create each node individually. 

int main() {
      int input[] = {3, 9, 20, -1, -1, 15, 7};
      struct TreeNode *root = treenode_build(input, ARRAY_SIZE(input));
      treenode_print(root);
      treenode_free(root);
      }

2D Array Helpers

These functions dynamically allocate and fill a 2D array with values from a static array of values.

int func(int **edges, int edgesSize, int *edgesColSize);

      int main() {
      int edges_input[6][2] = {{0, 1}, {0, 2}, {1, 3}, {1, 4}, {2, 5}, {2, 6}};
      struct two_d_arr *edges = two_d_arr_init(ARRAY_SIZE(edges_input), ARRAY_SIZE(edges_input[0]), edges_input);
      int res = func(edges.arr, edges.row_size, edges.col_size);
      two_d_arr_free(edges);
      }

If a problem's input has varying sized columns, one must create a array specifying each column's size and pass that to the function, instead of struct two_d_arr.col_size.
